# 🐱 Cat Math Adventure

A pixel art cat-themed math game website designed for 7-year-olds! Refer to Cambridge Grade 1 level.

## Features

- 🎮 **16 Math Units** covering essential math skills for 7-year-olds
- 🐱 **Pixel Art Cat Mascot** named "ALAINA" with animations
- ⭐ **Reward System** with stars and fish treats
- 🔊 **Sound Effects** (synthesized using Web Audio API)
- 📱 **Responsive Design** works on all devices

## 16 Math Units

1. **Counting Fun** - Count objects 1-20
2. **Addition Adventure** - Basic addition (1-10)
3. **Subtraction Safari** - Basic subtraction (1-10)
4. **Number Sequences** - Complete the pattern
5. **Even or Odd** - Identify even/odd numbers
6. **Greater or Less** - Compare numbers
7. **Addition with Pictures** - Visual addition
8. **Subtraction with Pictures** - Visual subtraction
9. **Make 10** - Number bonds
10. **Doubles** - Adding same numbers
11. **Shape Detective** - Identify shapes
12. **Pattern Party** - Continue the pattern
13. **Word Problems** - Math stories
14. **Money Math** - Count coins
15. **Time Teller** - Read o'clock times
16. **Final Challenge** - Mixed review

## How to Run

Simply open `index.html` in any modern web browser!

```bash
# Or use a simple HTTP server
python3 -m http.server 8080
# Then visit http://localhost:8080
```

## File Structure

```
cat-math-game/
├── index.html       # Main HTML structure
├── styles.css       # Pixel art styling
├── games.js         # All 16 game logic modules
├── cat-mascot.js    # Cat animations & interactions
└── sounds.js        # Sound effect synthesizer
```
